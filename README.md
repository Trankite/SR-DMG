## 📖 简介 Introduction
SR-DMG 是一款《崩坏：星穹铁道》伤害计算器，旨在为伤害计算提供便利。使用角色、光锥、遗器、敌人的模板快速完成配置，从外部快速导入面板数据，亦可以自定义遗器词条快速构建面板。
